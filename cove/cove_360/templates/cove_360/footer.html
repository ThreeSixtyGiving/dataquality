Data Quality Tool

The Data Quality Tool is a free tool which converts data provided using the 360Giving Data Standard format between JSON, Excel and CSV formats, checks that the data is valid and provides additional information about its usefulness or quality.

If you have questions or feedback, please raise an issue in Github.
